Prerequisites:     
1. Install nvm and node          
$ sudo apt update            
$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash       
$ nvm install v22.11.0  

2. Install node package dependencies    
$ npm install express   

To run:
1. nvm use v22.11.0
2. node server.js
