# Nole's Website

## Description
This repository represents a React app to host all of my website coding projects and
in future, perhaps, my resume and CV.

## Setup Instructions
1. Clone the repository     
```$ git clone https://github.com/NoleStites/nole_website.git```     
2. Enter the directory of the cloned repository      
```$ cd nole_website```      
3. Setup the React app       
```$ nvm use v22.11.0```     
```$ npm install```      
4. Start the app and view on port 3000      
```$ npm start```
